# 🏥 AI-Based Web Application for Anganwadi Workers

## 📌 Overview
This project is a full-stack web application designed to support Anganwadi workers in managing healthcare-related activities such as patient records, medical camps, and doctor interactions. The system improves accessibility and efficiency in rural healthcare management through a centralized digital platform.

---

## 🎯 Objectives
- Digitize patient record management for Anganwadi centers  
- Simplify medical camp scheduling and tracking  
- Enable doctor-patient interaction through an online portal  
- Improve healthcare service accessibility in rural areas  

---

## ⚙️ Features
- 👩‍⚕️ Doctor and Patient login portals  
- 🗂️ Patient record management (Create, Read, Update, Delete)  
- 📅 Medical camp scheduling and tracking  
- 📊 Organized storage of healthcare data  
- 🔐 Secure login authentication system  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  

---

## 🧩 System Modules
- **Admin Module:** Manage system data and users  
- **Doctor Module:** View patient details and manage medical camps  
- **Patient Module:** Register, view records, and access services  

---

## 🗄️ Database Design
- Structured relational database using MySQL  
- Efficient storage and retrieval of patient and medical data  
- Tables include:
  - Patients  
  - Doctors  
  - Appointments  
  - Medical Camps  

---

## ▶️ How to Run

1. Clone the repository:
```bash
git clone https://github.com/Dharshini11042004/your-repo-name.git
```
2. Move project to XAMPP/htdocs directory
3. Start Apache and MySQL using XAMPP
4. Import the database:
  - Open phpMyAdmin
  - Create a database
  - Import the .sql file
5. Run the application:

